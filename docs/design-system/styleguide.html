Design System · Calm Compliance

Internes Wissen, vertrauenswürdig zugänglich.

Das visuelle Fundament für Zweiplus Context. Aufgebaut auf der zweiplus Corporate Identity — das Signalgrün, eine editoriale Headline-Serife — und modernisiert für ein ruhiges, klares Produkt-Interface.

01

Farben

Das zweiplus-Grün (#6ec829) ist ein Signal, kein Dekor: es markiert Aktion, Erfolg und Fokus. Neutrale Töne sind dezent grün eingefärbt, damit das System aus einem Guss wirkt. Amber dient sparsam als Sekundärakzent.

Buttons

Primäraktionen nutzen das Signalgrün mit dunkler Tinte — punchy, modern und kontraststark (AA). Pro Ansicht nur eine primäre Aktion.
